# Observation Protocol — Calibrate

Purpose: structured in-context observation of a user evaluating an AI output.

Session length: 60 minutes

Flow:
- Consent and quick demographics (5 min)
- Assign a realistic, sanitized task and provide AI output in their tool (5 min)
- Think-aloud while evaluating (25–30 min)
- Probe for checks and heuristics (10 min)
- Code with breakdown taxonomy and close (5–10 min)

Observer checklist:
- Note time spent on each stage (Intent, Prompting, Generation, Interpretation, Evaluation, Action)
- Record specific verification actions (search, source check, calculation)
- Flag polish trap moments and heuristics used

Deliverable: affinity notes, timeline, heatmap of failure modes.
