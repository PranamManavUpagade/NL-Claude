# Interview Guide — Calibrate

Purpose: run semi-structured interviews to understand how users evaluate AI outputs and where trust breaks.

Duration: 45–60 minutes

Sections:
- Introduction & consent (5 min)
- Warm-up: tool usage + background (5 min)
- Task: give participant an AI output and ask them to evaluate (20–25 min)
  - Observe think-aloud
  - Prompt: "Please review this output and tell me whether you'd use it and why."
- Debrief: reasoning and verification steps (10 min)
- Wrap-up: suggestions and closing (5 min)

Core questions:
- What would you use this output for? (stakes, audience)
- How confident are you in the claims made? How did you decide?
- What checks would you run before using it?
- What is missing or unclear?
- Did anything feel overconfident or hedged?

Notes & coding:
- Use codes: OVER_TRUST, UNDER_TRUST, MISSED_GAP, POLISH_TRAP, EVAL_FATIGUE, TOOL_CONFUSION
- Record timestamps and quotes
