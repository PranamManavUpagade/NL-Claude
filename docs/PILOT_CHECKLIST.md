# Pilot Checklist — Calibrate

Goal: run a 4-week pilot (20–30 users) to measure CAR and collect qualitative feedback.

Pre-pilot
- [ ] Staging deploy of client and server
- [ ] Invite and consent flow prepared
- [ ] Analytics events instrumented (evaluation_start, evaluation_complete, trace_open, verdict_submitted)

Pilot week-by-week
- Week 1: onboard cohort, collect baseline surveys
- Week 2: monitor activation, push nudges
- Week 3: interviews with engaged users
- Week 4: 7-day follow-ups, compute CAR

Post-pilot
- [ ] Compile report with CAR, top fixes, UX observations
- [ ] Prioritize backlog for Phase 6
